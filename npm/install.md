# npm installieren

## Installation von npm

npm wird automatisch mit Node.js installiert.

1. **Node.js herunterladen:**  
   Gehe auf die offizielle Node.js-Webseite:  
   [https://nodejs.org/](https://nodejs.org/)

2. **Node.js installieren:**  
   Lade die passende Version für dein Betriebssystem herunter und installiere
   sie.  
   npm wird dabei automatisch mitinstalliert.

---

## Installation prüfen

Öffne ein Terminal (z.B. Eingabeaufforderung, PowerShell, Terminal) und prüfe
die Installation:

```bash
node -v
npm -v
```

Beide Befehle sollten eine Versionsnummer ausgeben.

---

## Zusammenfassung

- npm wird mit Node.js installiert.
- Nach der Installation stehen dir die Befehle `node` und `npm` zur Verfügung.
