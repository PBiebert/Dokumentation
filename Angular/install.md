# Angular CLI Installation und Nutzung

## Angular CLI installieren

Installiere die neueste Version der Angular CLI global:

```bash
npm install -g @angular/cli
```

Installiere eine bestimmte Version (z.B. 20.3.13):

```bash
npm install -g @angular/cli@20.3.13
```

---

## Angular CLI Version anzeigen

Zeige die aktuell installierte Version an:

```bash
ng version
```
